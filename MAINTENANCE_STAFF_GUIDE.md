## 🏠 Dashboard Overview

After logging in, you'll land on the **Maintenance Dashboard** which shows:

### Quick Stats
- **Open Orders**: Work orders waiting to be started
- **In Progress**: Work orders you're currently working on
- **Completed Today**: Tasks finished today
- **IPTV Devices**: Online/offline device count

### My Work Orders
- View all work orders assigned to you
- Color-coded by priority (Red=High, Yellow=Medium, Green=Low)
- Quick action buttons to start or complete tasks

### IPTV Device Status
- Real-time status of IPTV devices in hotel rooms
- Green dot = Online, Red dot = Offline
- Last seen time for each device

### Scheduled Maintenance
- Upcoming preventive maintenance tasks
- Due dates and equipment information
- Status indicators (Pending, Overdue, Completed)

### Tools & Parts Inventory
- Current stock levels
- Low stock alerts (red text)
- Quick "Request Parts" button

## 🛠️ Main Features

### 1. Work Orders

#### View All Work Orders
**Navigation**: Maintenance → Work Orders → All Orders

**What you can do**:
- See all maintenance requests
- Filter by status, priority, or category
- View room numbers and descriptions
- Track completion status

#### Open Orders
**Navigation**: Maintenance → Work Orders → Open

**What you can do**:
- View all pending work orders
- Accept assignments
- Start working on tasks

#### In Progress
**Navigation**: Maintenance → Work Orders → In Progress

**What you can do**:
- See tasks you're currently working on
- Update progress
- Mark as complete

#### Completed
**Navigation**: Maintenance → Work Orders → Completed

**What you can do**:
- View completed work order history
- Review completion notes
- Check completion times

---

### 2. IPTV Management

#### Devices
**Navigation**: Maintenance → IPTV → Devices

**What you can do**:
- Monitor all IPTV devices in the hotel
- Check online/offline status
- View device details (IP address, room number)
- See last activity time

**Device Status Indicators**:
- 🟢 Green = Online and working
- 🔴 Red = Offline or not responding
- ⏰ Last Seen = Time since last activity

#### Channels
**Navigation**: Maintenance → IPTV → Channels

**What you can do**:
- View available channel packages
- Check package pricing
- See active/inactive packages
- Understand room type assignments

**Package Types**:
- **Basic**: Standard channels for economy rooms
- **Premium**: Enhanced channels for deluxe rooms
- **VIP**: All channels for suite rooms

#### Troubleshoot
**Navigation**: Maintenance → IPTV → Troubleshoot

**What you can do**:
- See list of problematic devices
- Identify offline devices
- Access troubleshooting tools
- Track issue resolution

**Common Issues**:
- Device not responding → Check power and network
- No signal → Verify cable connections
- Buffering → Check network bandwidth

#### Installation
**Navigation**: Maintenance → IPTV → Installation

**What you can do**:
- View new devices awaiting installation
- See unassigned devices
- Complete room assignments
- Mark installations as complete

### 5. Time Tracking

**Navigation**: Maintenance → Time Tracking

**What you can do**:
- Clock in at start of shift
- Clock out at end of shift
- Track hours worked
- View today's time entries
- Start/complete tasks with time tracking

**Features**:
- Real-time clock display
- Hours worked today counter
- Tasks completed counter
- ON DUTY / OFF DUTY status
- Active task management
- Time entry history

**How to Use**:
1. Click "Clock In" when you start your shift
2. Start working on assigned tasks
3. Click "Start Task" when beginning a work order
4. Click "Complete Task" when finished
5. Click "Clock Out" at end of shift

## 📋 Common Workflows

### Starting Your Day
1. Log in to the system
2. Go to Dashboard to see overview
3. Click "Clock In" on Time Tracking
4. Review "My Work Orders" section
5. Start with high-priority (red) tasks first

### Handling a Work Order
1. Go to Work Orders → Open
2. Find your assigned task
3. Click "Start Work" button
4. Complete the maintenance work
5. Click "Complete Task"
6. Add completion notes if needed

### Reporting IPTV Issues
1. Go to IPTV → Devices
2. Find the problematic device
3. Check status and last activity
4. Go to IPTV → Troubleshoot
5. Follow troubleshooting steps
6. Update device status when fixed

### Requesting Parts
1. Go to Inventory → Request
2. Fill in item name
3. Specify quantity needed
4. Set priority level
5. Add notes about usage
6. Submit request
7. Wait for approval

### Checking Scheduled Maintenance
1. Go to Preventive → Calendar
2. View upcoming tasks
3. Note due dates
4. Plan your schedule accordingly
5. Complete tasks before due date
6. Mark as complete when done

## 🎨 Understanding Color Codes

### Priority Colors
- 🔴 **Red Background**: High priority - urgent
- 🟡 **Yellow Background**: Medium priority - important
- 🟢 **Green Background**: Low priority - routine

### Status Colors
- 🟢 **Green Badge**: Completed/Online/Active
- 🟡 **Yellow Badge**: Pending/In Progress
- 🔴 **Red Badge**: Overdue/Offline/Error
- ⚪ **Gray Badge**: Inactive/Cancelled

### Stock Level Colors
- 🟢 **Green Text**: Good stock level
- 🟡 **Yellow Text**: Low stock warning
- 🔴 **Red Text**: Critical - reorder needed

## 🆘 Troubleshooting

### Can't Log In
- Check username/password
- Ensure caps lock is off
- Clear browser cache
- Contact IT support

### Page Not Loading
- Refresh the page (F5)
- Clear browser cache
- Check internet connection
- Try different browser

### Can't See Work Orders
- Check if you're assigned to tasks
- Verify your role permissions
- Contact supervisor
- Refresh the page

### Time Tracking Issues
- Ensure you're clocked in
- Check system time
- Verify task assignment
- Contact supervisor
